## 032 - Inicialização com chaves { }

- *em breve*

### Pontos

- Brace-initialization !

- Mais segurança na Inicialização das variáveis

### Exemplos

Com chaves:

```cpp
#include <iostream>

int main()
{
    using std::cout;
    using std::endl;

    // experimente trocar o tipo aqui!
    // char, short, int, long, long long, float, double, long double
    int value;

    std::cin >> value;

    char c{value};
    short sh{value};
    int i{value};
    long l{value};
    long long ll{value};
    float f{value};
    double d{value};
    long double ld{value};

    cout << c << '\n'
         << sh << '\n'
         << i << '\n'
         << l << '\n'
         << ll << '\n'
         << f << '\n'
         << d << '\n'
         << ld << endl;

    return 0;
}
```
